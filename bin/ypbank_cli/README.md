# Пакет консольных утилит для работы с транзакциями

Содержит две утилиты командной строки для сравнения файлов с транзакциями (comparer) и для конвертации транзакций между различными форматами (converter).

# Comparer

Утилита командной строки для сравнения двух файлов с транзакциями.

Позволяет сравнивать файлы, сохраненные в различных форматах (CSV, текст, бинарный), и находить расхождения в наборах транзакций.

## Использование

```bash
comparer --file1 <ПУТЬ_К_ФАЙЛУ_1> --file1-format <ФОРМАТ_1> --file2 <ПУТЬ_К_ФАЙЛУ_2> --file2-format <ФОРМАТ_2>
```

### Аргументы

*   `--file1`: Путь к первому файлу для сравнения.
*   `--file1-format`: Формат первого файла.
*   `--file2`: Путь ко второму файлу для сравнения.
*   `--file2-format`: Формат второго файла.

### Поддерживаемые форматы

Для аргументов `--file1-format` и `--file2-format` допустимы следующие значения:

*   `csv`
*   `txt`
*   `bin`

## Примеры

Сравнение CSV файла с бинарным файлом:

```bash
comparer --file1 data.csv --file1-format csv --file2 data.bin --file2-format bin
```

Сравнение двух CSV файлов:

```bash
comparer --file1 old_data.csv --file1-format csv --file2 new_data.csv --file2-format csv
```

## Вывод

Утилита выведет в стандартный поток вывода (stdout) информацию о различиях:

*   `Files are identical`: если наборы транзакций полностью совпадают.
*   `Transaction with id <ID> is only in file 1`: если транзакция есть в первом файле, но отсутствует во втором.
*   `Transaction with id <ID> is only in file 2`: если транзакция есть во втором файле, но отсутствует в первом.

# Converter

Утилита командной строки для конвертации файлов с транзакциями между различными форматами.

Позволяет преобразовывать данные из одного поддерживаемого формата (CSV, TXT, BIN) в другой. Результат конвертации выводится в стандартный поток вывода (STDOUT), что позволяет перенаправлять его в файл.

## Использование

```bash
converter --input <ПУТЬ_К_ФАЙЛУ> --input-format <ИСХОДНЫЙ_ФОРМАТ> --output-format <ЦЕЛЕВОЙ_ФОРМАТ> > <ВЫХОДНОЙ_ФАЙЛ>
```

### Аргументы

*   `--input`: Путь к исходному файлу для конвертации.
*   `--input-format`: Формат исходного файла.
*   `--output-format`: Желаемый формат выходных данных.

### Поддерживаемые форматы

Допустимые значения для `--input-format` и `--output-format`:

*   `csv` — Comma-Separated Values
*   `txt` — Текстовый формат "ключ-значение"
*   `bin` — Специализированный бинарный формат

## Примеры

Конвертация CSV файла в бинарный формат:

```bash
converter --input data.csv --input-format csv --output-format bin > data.bin
```

Конвертация бинарного файла в текстовый формат:

```bash
converter --input data.bin --input-format bin --output-format txt > data.txt
```

Конвертация текстового файла в CSV:

```bash
converter --input data.txt --input-format txt --output-format csv > data.csv
```

## Ограничения

*   Входной и выходной форматы не должны совпадать.
*   Входной файл должен существовать и быть доступным для чтения.
